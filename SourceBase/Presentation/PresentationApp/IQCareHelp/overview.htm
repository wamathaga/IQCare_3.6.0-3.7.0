IQCare was initially designed for the international HIV patient care community and has grown into a fully functional HIMS system supporting HIV, TB, PMTCT and other health care areas. New features continue to be specified directly by the Local Partners and Strategic Information Advisers in the field, creating a high value system.

 

IQCare (browser-based data entry application)

•IQCare application configuration

•HIV Care Data entry

•Link to IQTools reports and patient level reports

 

oA robust underlying database which is excellent for reporting

oBusiness rule driven clinical forms with data validation

oPatient appointment scheduling

oHIV Care Static form set, MOH Static form set (MOH HIV Care card) and Smart ART dynamic forms (created in using IQCare Management form builder)

oA variety of patient, facility and donor reports as well as a flexible custom reports module

oCustomizable lists that display within the forms

oCustom fields that can be added to static forms

oForm builder to build your own forms associated with new program/service areas

oFine grain user access features that manages the areas a user can or can not access within the application.

oThe ability to deploy IQCare as a stand alone application or as a LAN application

 

IQCare Management (Windows based form configuration)

•Form builder service areas, fields and forms

•Supply Chain Management configuration and inventory management
