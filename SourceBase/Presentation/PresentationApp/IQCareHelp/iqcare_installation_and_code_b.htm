IQCare Installation and Code Base

  

IQCare is an open source product and is freely available to download and install. The full code base and the installation executable can be found on Codeplex, an website for .Net open source applications. http://fgiqcare.codeplex.com/

 

•Source Code tab - is where you can find the entire IQCare source code

•Downloads tab - is where you can find the full install executable and the update installations.

•Documentation tab - is where you can find trainings and user guides

 

The full installation come with or direct you to pull down from the web all of the prerequisites to run IQCare including SQL Express. The instructions are for Windows 7, 8 and 8.1.
