Login Credentials

Before you can login, your IQCare administrator must set up a Username and Password for you. The areas of the application that you can navigate to are dependent on the access rights that your administrator has given to you. The navigation bar will display only those areas in the application that you have rights to enter.

 

You will be required to change your password on the first login.

 

Strong Passwords

The system may be set up to require a strong password. See Facility/Satellite Set up. If the application has been configured to use strong passwords, the password must meet these requirements:

•Minimum of 6 characters in length

•At least one upper case letter

•At least one numeric character

•At least one non alpha character

•No more than 3 consecutive characters e.g. ‘1234’ or ‘abcd’

•You may not use the work ‘password’, the user’s first name, last name or user name

 

Expiring Passwords

The system may be set up to require the user to reset their password base on the number of days configured in the Facility/Satellite Set up page. When the user logs in and it's time to change the password, IQCare redirects the user to the change password page.

 

Multi-attempt Lockout

The system will be locked if a User attempts to enter IQCare 3 times with an incorrect Username or Password. The system will allow the User to login after 10 minutes.

 

Facility/Satellite Selection

You are required to login to IQCare in order to enter patient data. In addition to the Username and Password, you must also specify the Preferred Location for which you want to enter patient data. Patients not located at the clinic/satellite you have specified at login can not be accessed; you must log out and re-login and choose the Preferred Location to access patient data at the location they are receiving treatment.

 

Preferred Location Check Box

For some IQCare configurations, the list of Facility/Satellite location can be very long. A data entry clerk will only want to enter data for patients getting treatment at a handful of locations. These “Preferred Locations" are set in the Facility/Satellite configuration and displayed when the Preferred Location box is checked.

 

Login Image

The login page image can be customized through the Facility Set Up page.  For example, you can have a logo of your facility as the login image.
