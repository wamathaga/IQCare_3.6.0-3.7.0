Navigation:  Administration > Custom Lists >

Update an Entry in a  Custom List

1.From the navigation bar, select Customize Lists

2.Select the list you would like to modify by clicking anywhere in the row with the list name. The list entries display in a table in priority order.

 

3.Click on the list entry to update. Note that for restricted lists, not all of the entries can be updated as indicated by the grayed-out Update button.

 

 

4.All entries added by you that are displayed after the restricted portion of the list can be updated by changing the entry text, status or priority.

5.Click Update

6.The list entry will be updated
