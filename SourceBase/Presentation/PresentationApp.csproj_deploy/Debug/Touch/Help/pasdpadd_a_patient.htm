Add a New Patient

1.From the Find/Add screen, click on the Add Patient to display the Add Patient screen.

2.Enter the patient's first name, last name, sex, date of birth and folder number. All of these fields are required (with the middle name the only field that is not required.) Click the Add button to add the patient to the system.

3.Once you have clicked the Add button, you will automatically be taken directly to the Registration form. The data you just entered will be displayed.

 

Note: once you have saved a patient with a folder number, you cannot change the folder number. When the system in integrated with the hospital's Delta 9 patient registration system, IQCare-PADMT will check the name, sex, DOB and folder number to verify that all of the data matches. If not, the system will display a user error.
