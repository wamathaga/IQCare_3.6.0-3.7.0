Administration

IQCare-PADMT leverage IQCare Standard's administration features.

 

IQCare-PADMT administration features:

•Custom lists - modify the default selects list displayed n clinical forms

•Backup database - set up a daily automated back up or manually run a back up. Manually run a de-identified back up.

•Configure the care ended form - modify the default the care ended form

•Configure the audit trail - set up the audit trail

 

To access the IQCare's administration, click on the More...icon and select the Administration link.
